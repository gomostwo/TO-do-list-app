# Context Map

## Project Type

- **Framework**: None (Vanilla HTML/JS/CSS)
- **Language**: JavaScript
- **Package Manager**: npm (not yet initialized, if needed later)

## Important Folders

- `index.html` — The main entry point and UI for "My Day: To-Do List Home"
- `src/js/` — App state management and user interaction logic
- `src/css/` — Custom styling (supplementing TailwindCSS)
- `mockups/` — Original HTML designs downloaded from Google Stitch
